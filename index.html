About Me

I’m Sathish Kumar S, a Full Stack Developer specializing in the MERN stack, focused on building scalable and user-friendly web applications.

I am a passionate Frontend Developer with a strong foundation in HTML, CSS, JavaScript, Bootstrap, and React. I enjoy building responsive, user-friendly web applications and continuously improving my development skills.

I have developed several projects, including Zyntra Store using HTML and CSS, Azure Palms Resorts using Bootstrap, and a JavaScript-based web project for Vdev Technologies for a startup company. I am currently building an UrbanCart, a React-based eCommerce application featuring reusable components, product filtering, cart functionality, and responsive design, where I am gaining hands-on experience with reusable components, state management, and modern frontend development practices.

Alongside frontend development, I am expanding my full-stack knowledge by learning Node.js, Express.js, MongoDB, and MySQL to build complete web applications. I am eager to contribute to a professional development team, learn from experienced engineers, and grow as a software developer.
